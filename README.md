## 🛠️ How to Use

### View Locally

1. Clone the repo:
   ```bash
   git clone https://github.com/hsinidev/1000-ui-projects.git
   cd 1000-ui-projects
   ```

2. Open any project's `index.html` in your browser:
   ```bash
   # Example:
   start "FinTech & Banking/stitch_neovault_digital_banking/index.html"
   ```

### Live Preview (with VS Code)
Install the **Live Server** extension and right-click any `index.html` → *Open with Live Server*.

## 🎨 Design System & Technologies

| Technology | Usage |
|------------|-------|
| **HTML5** | Semantic structure & accessibility |
| **CSS3** | Custom properties, animations, glassmorphism |
| **JavaScript (ES6+)** | Interactivity & dynamic UI |
| **CSS Grid & Flexbox** | Responsive layouts |
| **CSS Custom Properties** | Design tokens & theming |
| **Google Fonts** | Premium typography |

### Design Principles Applied
- 🌑 **Dark-first Design** — Most projects feature dark/midnight themes
- 🔮 **Glassmorphism** — Frosted glass panels and blurred backgrounds
- ✨ **Micro-animations** — Hover effects, transitions, and scroll animations
- 📐 **Grid Systems** — Consistent 8px/12px grid-based layouts
- 🎨 **Color Psychology** — Industry-specific color palettes
- ♿ **Accessibility** — ARIA labels, contrast ratios, keyboard navigation
