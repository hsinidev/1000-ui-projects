Maintenance of Moving Parts
expand_more

Precision hinges require quarterly lubrication using non-corrosive silicone oils. Avoid WD-40 or solvent-based sprays which can degrade the micro-washers.

DANGER: Excessive force on wing-doors may result in pivot failure.
