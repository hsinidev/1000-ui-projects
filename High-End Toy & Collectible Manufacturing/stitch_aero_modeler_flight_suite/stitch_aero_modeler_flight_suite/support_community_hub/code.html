SECTION 01: TECHNICAL SPECIFICATIONS expand_more
How do I calibrate the RC-AFM v4 gyro-stabilizer? keyboard_arrow_down
Access the 'Diagnostics' tab in the Command Center. Connect your airframe via high-speed USB-C. Ensure the craft is on a perfectly level carbon-fiber or slate surface. Select 'Auto-Level' and wait for the triple LED confirmation (Amber-Green-Green).
Firmware Update Failure: Error Code 0x992 keyboard_arrow_down
This error usually indicates a signal parity mismatch. Reset your local hub and ensure the AERO-X proxy is bypassed. If the issue persists, perform a hard-reset by holding the 'Sim Hub' manual override for 5 seconds.
SECTION 02: LOGISTICS & SHIPPING expand_more
Global Logistics: LiPo Battery Restrictions keyboard_arrow_down
High-capacity batteries are classified as Class 9 Dangerous Goods. We utilize specialized temperature-controlled aerospace freight for all overseas Pilot Kits to ensure cell integrity upon arrival.
SECTION 03: REGULATORY COMPLIANCE expand_more
Remote ID Integration Requirements keyboard_arrow_down
All AERO-X airframes are pre-fitted with FAA-compliant RID broadcast modules. Activation occurs automatically upon first GPS lock in the Simulator Hub.
