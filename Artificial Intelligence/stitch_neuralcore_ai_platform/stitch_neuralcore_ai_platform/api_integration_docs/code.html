Harness the power of high-precision neural architectures. Our API provides direct access to optimized latency-critical inference endpoints.

POST

/v2/inference/deploy

Deploys a pre-trained neural weight set to a dedicated high-compute edge node. Requires architect_token.

Headers
Authorization Bearer <TOKEN>
Content-Type application/json
cURL Node.js Python 
curl -X POST "https://api.neuralcore.ai/v2/inference/deploy" \
  -H "Authorization: Bearer $ARCHITECT_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "model_id": "nc-synapse-8b",
    "region": "us-hyper-01",
    "scaling": {
      "min": 2,
      "max": 10,
      "trigger": "latency_ms > 45"
    }
  }'
