API Protection

How do I rotate production JWT secrets without downtime? expand_more
Use the /v1/auth/rotate endpoint with a dual-key grace period. AppSecure supports maintaining both previous and current secrets for a 24-hour overlap.
Can I enforce rate limits based on specific header metadata? expand_more
Yes. Configure custom "Identifier Keys" in the Shield Policy panel to use X-App-Tenant-ID or any other custom header.
What is the latency overhead of the AppSecure proxy? expand_more
AppSecure adds <2ms of latency for standard header inspection and <15ms for deep packet inspection of payload bodies.

Bot Mitigation

Distinguishing between 'Good' and 'Bad' bots? expand_more
Our heuristic engine uses fingerprinted behavior patterns. Search engines (Google/Bing) are automatically whitelisted via verified IP checks.
Configuring CAPTCHA-less mitigation? expand_more
Enable "Proof of Work" challenges in the Dashboard. This forces the client to perform a small computation, deterring low-cost bot nets without user friction.
