01

How do I authorize a new NODE_AEGIS hardware bridge?

Navigate to the Admin Panel > Hardware Tokens. Generate a 256-bit RSA key and physically inject it via the encrypted USB-C port on the front panel of the bridge. Ensure the LED status is solid Amber before proceeding.

$ aegis-cli --provision --token [ID_HASH] --local-only
02

Resolving 'ERR_PROTOCOL_MISMATCH' on Modbus registers?

This typically occurs when the endianness of the controller doesn't match the Aegis parser settings. Open the device profile and toggle the 'Invert Byte Order' flag.

03

Emergency System Isolation Procedure (Air-Gap)

In the event of a breach, trigger the 'LOCKDOWN' command from the dashboard or physically press the red emergency button on the AEGIS core unit. This severs all external TCP links.
