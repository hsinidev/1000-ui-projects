Operating & Maintenance Instructions

Revised: Sept 1968

Section I: Quick Start

01

Connect the primary power source to the rear IEC socket. Ensure the toggle switch is in the DOWN (Off) position before applying power.

02

Allow the 12AX7 vacuum tubes exactly 60 seconds to reach operating temperature. The amber pilot lamp will pulse until stable.

03

Slowly rotate the INPUT GAIN bakelite knob clockwise until the VU meter needle begins to deflect between -3 and 0 VU.

Section III: Knob Functions

Input Gain

Controls the level of the primary input signal. Higher settings introduce harmonic saturation and "tube warmth."

Low Cut (Hz)

A 12dB/octave high-pass filter. Set to 'OFF' for full range response. Positions: 20Hz, 40Hz, 80Hz.

Phase Invert

Switches the absolute polarity of the output. Use when combining multiple microphones on a single source.
