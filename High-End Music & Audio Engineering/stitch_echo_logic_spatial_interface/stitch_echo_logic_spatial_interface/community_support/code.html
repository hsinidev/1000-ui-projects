Technical FAQ

Optimizing Ray-Tracing for VR?

expand_more

To optimize spatial audio ray-tracing, ensure your geometry primitives are simplified. High-poly models significantly increase computational load for reflection calculations.

HRTF Personalization in SDK v.01

expand_more

Spatial Core v.01 supports SOFA file imports for personalized Head-Related Transfer Functions. You can upload these in the Mapping tab under Calibration settings.

Latency Thresholds for AR?

expand_more

Ideal motion-to-sound latency should be under 20ms. Echo-Logic uses a predictive buffer system to maintain spatial coherence during rapid head movements.

Binaural Rendering Engine

expand_more

Our engine uses third-order Ambisonics as the base layer, ensuring that dynamic sound sources maintain high fidelity during rotation.
