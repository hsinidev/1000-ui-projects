How do I reset the E-Stop telemetry lock? expand_more
To reset the lock, navigate to System Health > Diagnostics > Manual Override. Input your technician ID and press "Telemetry Flush". Ensure unit is stationary.
Recommended blade angle for wet rye grass? expand_more
We recommend a 3.5-degree upward pitch with a rotation speed of 2,800 RPM. This prevents clumping while maintaining a clean shear.
