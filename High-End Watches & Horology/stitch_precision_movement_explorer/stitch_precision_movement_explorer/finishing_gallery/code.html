TECHNIQUE 01

Geneva Stripes

Standard: Côtes de Genève

A series of parallel wave-like patterns applied to the visible bridges of the movement. This technique requires an abrasive wooden tool to create a rhythmic oscillation that disperses light in a mesmerizing arc.

Edge Finish: Hand-Polished Bevels

The removal of sharp edges from movement components, replaced by a 45-degree chamfer that is hand-polished to a mirror finish. This creates a brilliant highlight that separates individual components visually.
