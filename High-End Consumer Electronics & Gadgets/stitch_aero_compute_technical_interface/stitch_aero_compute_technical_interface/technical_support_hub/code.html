SYSTEM_FAQ

How do I verify checksum integrity of AERO-OS? expand_more
Use the terminal command `aero-verify --checksum [path]` to compare against the master hash provided in your deployment manifest. Any discrepancy will automatically lock the bootloader for security.
What are the critical temperature thresholds? expand_more
Core-Alpha units are rated for continuous load up to 85°C. At 92°C, the system triggers stage-1 throttling. Hardware shutdown occurs at 105°C to prevent silicon degradation.
How to recalibrate telemetry gauges? expand_more
Navigate to Maintenance -> Sensor Array -> Recalibrate. The system will perform a 30-second sweep. Ensure no heavy computational loads are running during this window.
