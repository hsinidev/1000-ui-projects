How do I export high-resolution chromosome heatmaps for publication?

Users can navigate to the 'Visualization' tab in the Sequence Browser. Select 'Export' and choose 'Publication Grade (300 DPI)'. File formats supported include TIFF, SVG, and high-bitrate PDF. Ensure color-blind accessible palettes are selected if required by your journal.
