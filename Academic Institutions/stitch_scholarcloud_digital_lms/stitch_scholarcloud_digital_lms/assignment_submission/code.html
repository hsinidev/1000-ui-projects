Your task is to develop a Python-based implementation of the A* search algorithm optimized for dynamic graph weights. The submission must include:

  • A well-documented source file (.py or .ipynb).
  • A performance analysis report (PDF) comparing your implementation against standard Dijkstra.
  • A visualization of the search path on the provided 'Logistics-Set-B' dataset.

Note: All external libraries must be specified in a requirements.txt file.
