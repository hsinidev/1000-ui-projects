Popular Questions

How do I invite my team? expand_more
Navigate to the "Admin" tab in your sidebar, then select "Members". You can invite team members via email or by sharing a unique workspace link. SynthSpace supports tiered permissions for seamless collaboration.
Connecting VR hardware expand_more
Connect your headset via the desktop app. SynthSpace supports Oculus Quest 2, Pro, and Valve Index natively. Ensure your firmware is up to date for the best floating HUD experience.
Data vault encryption expand_more
All files in the Vault are secured with AES-256 end-to-end encryption. Only authenticated workspace members with specific keys can decrypt and view sensitive documents.
Exporting spatial data expand_more
Use the "Files" export tool to download spatial metadata in .JSON or .OBJ formats. This allows for easy integration with external 3D modeling software and productivity suites.
