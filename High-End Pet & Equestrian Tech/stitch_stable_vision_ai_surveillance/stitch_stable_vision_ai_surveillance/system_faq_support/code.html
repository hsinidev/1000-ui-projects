How do I reset the thermographic drift on external sensors? expand_more

Navigate to Admin Panel > Hardware > Calibration. Initiate the 'Thermal Baseline' routine during off-peak hours (02:00-04:00) to ensure zero atmospheric interference. Total cycle time is 180 seconds.

CMD: /sys/calibrate --mode thermal --force
Lidar interference detected in high-humidity conditions. expand_more

High moisture density can trigger false positive occlusion alerts. Adjust the 'Refraction Offset' in the sensor configuration file to 0.45μm. If persistence continues, physical lens heating must be engaged.
