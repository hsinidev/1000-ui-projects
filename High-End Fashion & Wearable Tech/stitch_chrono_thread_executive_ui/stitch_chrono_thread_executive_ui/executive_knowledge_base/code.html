Garment Care

REVISED: OCT 2024
How do I safely clean the electronic lapel sensors? expand_more

Sensors should be cleaned using a lint-free micro-fiber cloth lightly dampened with 70% Isopropyl alcohol. Avoid direct immersion. For general fabric care, use specialized dry cleaning services certified for wearable tech.

Can the suit withstand heavy rainfall? expand_more

The Chrono-Thread series is IPX4 rated. It is resistant to splashes and light rain, but heavy exposure may compromise the haptic feedback modules in the cuffs. Always air-dry at room temperature.

Command Cuff Calibration

STATUS: ACTIVE
Re-syncing haptic patterns after a firmware update expand_more

Initiate a hard reset by holding the primary cuff button for 10 seconds. Open the SMART_SUITE app and navigate to "Device Sync" to recalibrate the three-tier haptic intensity levels.

NFC Programming

VERSION 2.4.0
Linking multiple digital profiles to a single sleeve expand_more

Users can map up to 4 distinct NFC payloads using the "Contextual Swap" feature. Tap the cuff twice to cycle between Professional, Social, Security, and Smart Home profiles.
