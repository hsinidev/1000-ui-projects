1.0 EXECUTIVE SUMMARY

The NACA 0012 airfoil is a symmetrical 4-digit NACA airfoil. This document details the numerical validation of flow characteristics under Mach 0.15 to 0.8 conditions. Current simulations utilize the k-ω SST turbulence model with a mesh density of 4.2M cells.
