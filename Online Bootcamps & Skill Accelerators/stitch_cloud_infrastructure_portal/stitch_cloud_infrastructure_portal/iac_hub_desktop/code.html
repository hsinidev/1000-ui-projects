Zero Trust Gateway

This module implements a defense-in-depth security perimeter for AWS workloads. It automatically configures:

  • CloudFront Distribution with AWS WAFv2 association
  • Custom IP Rate Limiting and Geo-blocking
  • Automated TLS Certificate via ACM
  • Deep packet inspection rules for common vulnerabilities

Prerequisites

Requires an existing Route53 Hosted Zone ID for DNS verification.

CODE PREVIEW (HCL) 
module "security_gateway" {
  source  = "./modules/zero-trust"
  version = "1.1.2"

  # Network Config
  vpc_id     = var.vpc_id
  env        = "production"
  waf_rules  = ["sql_injection", "xss"]
}

Approx. provision time: 4m 30s
