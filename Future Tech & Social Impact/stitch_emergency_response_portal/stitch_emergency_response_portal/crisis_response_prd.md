# Crisis Response Platform - PRD

## Overview
A high-utility, accessibility-first platform designed for rapid crisis response.

## Key Features
- **High-Contrast 'Emergency' Palette**: Designed for maximum visibility and accessibility (WCAG AAA).
- **Low-Bandwidth Mode**: A toggle to simplify the UI, remove heavy assets, and optimize for slow internet connections.
- **Interactive Resource Map**: Real-time status markers for shelters, food, medical aid, etc.
- **Voice-Search**: Clear, multi-language voice-activated search for hands-free or vision-impaired utility.

## Screens to Generate
1. **Dashboard (Main)**: Overview of critical alerts, voice search, and low-bandwidth toggle.
2. **Resource Map**: Full-screen interactive map with real-time status markers and filters.
3. **Emergency Alerts**: A prioritized list of active crises and safety instructions.
4. **Resource Directory**: A searchable, text-heavy directory of aid organizations and contact info (optimized for Low-Bandwidth).
