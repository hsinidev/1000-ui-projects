01 Recalibrating Ultrasonic Sensors in Saltwater Environments expand_more
Ensure the transducer face is clear of bio-fouling. Use the CAL-99 protocol via the operator terminal. If salinity levels exceed 35 PSU, apply the density correction coefficient as specified in the Technical Manual Section 4.2.
02 Emergency Valve Override (Manual Lock-Out) expand_more
In the event of electronic signal failure, rotate the mechanical handwheel clockwise until the hard stop is reached. Engagement of the lock-out pin requires a Grade 5 security key. Monitor the local pressure gauge for physical confirmation.
