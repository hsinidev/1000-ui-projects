Most Popular

Gastric Sleeve

Also known as sleeve gastrectomy, this procedure reduces the stomach size by about 80%, limiting food intake and reducing hunger hormones.

  • done Minimally invasive approach
  • done Typical recovery: 2-3 weeks
  • done Average weight loss: 60-70% excess weight

Gastric Bypass

Roux-en-Y Gastric Bypass creates a small stomach pouch and reroutes the digestive tract, providing both restrictive and malabsorptive weight loss.

  • done Gold standard for long-term weight loss
  • done Excellent for resolving Type 2 diabetes
  • done Average weight loss: 70-80% excess weight
