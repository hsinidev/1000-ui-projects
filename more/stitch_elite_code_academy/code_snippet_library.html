Retry with Jitter

bookmark

Exponential backoff algorithm with full jitter implementation to prevent thundering herd problems in distributed systems.

content_copy COPY 
const retryWithJitter = (attempt) => {
  const backoff = Math.pow(2, attempt) * 1000;
  const jitter = Math.random() * backoff;
  return Math.min(30000, backoff + jitter);
};

Intersection Observer Hook

USED BY 1.2K DEVS bookmark

A highly optimized, reusable React hook for lazy loading, infinite scroll, and scroll-triggered animations. Handles cleanup and browser compatibility out of the box.

  • check_circle Zero Dependencies
  • check_circle Typescript Support

content_copy COPY 
function useIntersectionObserver(ref, options) {
  const [isIntersecting, setIntersecting] = useState(false);

  useEffect(() => {
    const observer = new IntersectionObserver(([entry]) => {
      setIntersecting(entry.isIntersecting);
    }, options);

    if (ref.current) observer.observe(ref.current);
    return () => observer.unobserve(ref.current);
  }, [ref, options]);

  return isIntersecting;
}
