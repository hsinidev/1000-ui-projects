Troubleshooting Guides

Root Cause Analysis

Jitter typically occurs when the Master Clock node loses sync with the stadium-wide Dante or AVB network. This manifests as "clicks" or "pops" in the high-frequency drivers.

Step 1: Verify all fiber-optic links are seated with < 0.5dB loss.

Step 2: Force re-sync via the Node_01 Terminal using the --sync-hard command.

Step 3: Switch to Secondary Clock if jitter persists above 10 picoseconds.
